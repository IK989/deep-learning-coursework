# deep-learning-coursework

This repository contains practical implementations completed as part of the Deep Learning course in the M.Sc. Artificial Intelligence program at FAU Erlangen–Nürnberg.

The repository demonstrates the implementation of fundamental deep learning concepts using Python and NumPy, progressing from neural network basics to convolutional and recurrent neural networks.

## Topics Covered

- Python and NumPy Fundamentals
- Data Generation and Preprocessing
- Fully Connected Neural Networks
- Backpropagation
- Convolutional Neural Networks (CNNs)
- Optimization Algorithms
- Regularization Techniques
- Recurrent Neural Networks (RNNs)

## Repository Structure

- Exercise 1 – Python & NumPy Fundamentals
- Exercise 2 – Fully Connected Networks
- Exercise 3 – CNNs and Optimization
- Exercise 4 – Regularization and Recurrent Neural Networks


These implementations were developed as part of academic coursework for educational purposes.
